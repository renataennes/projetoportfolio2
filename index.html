Projetos em Ciências de Dados

Desenvolvimento de um Painel Gerencial para negócio com o Streamlit

Neste projeto, os conceitos de programação em Python, manipulação de dados, planejamento de solução, associado com ferramentas de desenvolvimento web (Streamlit e Github), foram usados para desenvolver um painel com as principais métricas de uma empresa fictícia de compra e venda de imóveis.

O resultado final do projeto foi um painel hospedado em ambiente Cloud e disponível por meio de link web. O painel pode ser acessado por qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram

  • Python
  • VS Code
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • Github
